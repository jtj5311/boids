# New Laptop Setup Notes — 2026-02-12

## What the .gitignore excludes

- `**/target/` — Rust build artifacts. Regenerated by `cargo build`.
- `**/__pycache__/` — Python bytecode cache. Regenerated automatically on run.
- `.claude*` — Claude Code session/config files. Not needed to run the project.

None of these are needed to run the project on a new machine.

## Setup on a new laptop

### Prerequisites

- Python + pip
- Rust + Cargo

### Python (leah/jax)

```bash
cd leah/jax
pip install -r requirements.txt
```

### Rust (leah/rust)

```bash
cd leah/rust
cargo build
```
